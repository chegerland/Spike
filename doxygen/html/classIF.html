Returns a spike train, where IF is subject to a get_value.

Produces a spike train, i.e. an array with as many steps as the time frame, that contains a zero if the neuron has not spiked and a 1 if it has spiked in the corresponding time bin. We use the Euler-Maruyama-Scheme to solve the SDE for the IF neuron an simply put a 1 into the spike train array, whenever the neuron spikes.
